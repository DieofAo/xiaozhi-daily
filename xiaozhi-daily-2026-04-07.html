NEWARM: Advantage Reward Modeling for Long-Horizon Manipulation
Robotics 奖励建模
长视距机器人操作任务中稀疏奖励让 RL 难以分配信用。ARM 提出了一种新思路：不建模绝对进度，而是估计相对优势。核心创新是"三态标注"策略——将动作标为"渐进/回退/停滞"三类，极大降低人类标注负担并保持高一致性。在离线 RL 管线中实现自适应动作-奖励重加权，过滤次优样本。在高难度长视距毛巾折叠任务上达到 99.4% 成功率，显著超越当前 VLA 基线，且几乎无需人工干预。对 VLA 后训练范式有重要参考价值。

EUPE: Efficient Universal Perception Encoder
Multimodal 视觉编码器
Meta FAIR 提出的 EUPE 系列是面向边缘设备的紧凑型通用视觉编码器，参数量 低于 100M，却在图像理解、密集预测和 VLM 任务上媲美甚至超过同等大小的领域专家模型。核心方法是"先扩后缩"知识蒸馏：先从多个专家教师蒸馏到一个大型代理教师，再从该单一教师缩放为高效编码器。已开源代码和模型（GitHub + HuggingFace）。对端侧 VLA 部署有重要参考意义——为机器人系统提供了一个体积小、能力强的统一视觉前端选项。

HOLOS: LLM-Based Multi-Agent System for Building Agent Networks
Agent 多智能体协作
HOLOS 提出了一种基于 LLM 的多智能体系统架构，核心目标是构建可组合的"智能体网络"（Agent Networks）。不同于传统的单体 Agent 或简单的链式调用，HOLOS 将多个异构 Agent 组织为动态协作网络，支持灵活的任务分配和信息共享。该框架有望推动多 Agent 在复杂机器人系统（如多机协同操作、仓储调度等）中的应用。
