VLA-OPD: Bridging Offline SFT and Online RL for Vision-Language-Action Models via On-Policy Distillation HOT
arXiv 2603.26666 · 2026-03-27 VLA 在线策略蒸馏
提出在线策略 VLA 蒸馏框架 VLA-OPD，连接离线 SFT 的效率与在线 RL 的鲁棒性。核心思路是让专家教师对学生自生成的轨迹提供密集的 Token 级监督，实现策略诱导状态上的主动错误纠正。关键创新在于采用反向 KL 目标——既避免前向 KL 的模式覆盖熵爆炸，也避免硬交叉熵的过早熵崩溃，通过过滤教师认知不确定性同时保持动作多样性。在 LIBERO 和 RoboTwin2.0 上显著提升 RL 样本效率和 SFT 鲁棒性，有效缓解后训练灾难性遗忘。

Ruka-v2: Tendon Driven Open-Source Dexterous Hand with Wrist and Abduction for Robot Learning HOT
arXiv 2603.26660 · 2026-03-27 Robotics 灵巧手硬件
开源肌腱驱动灵巧手 Ruka-v2，相比 v1 新增解耦的 2 自由度并联手腕和手指外展/内收功能。并联手腕提供独立的屈伸和桡/尺偏，便于在狭窄环境（如橱柜内部）操作；外展功能支持抓取薄物体、手内旋转和书法等精细动作。通过远程操作用户研究评估，相比 v1 完成时间减少 51.3%，成功率提升 21.2%。演示了 13 项双臂/单臂灵巧任务及 3 项自主策略学习。所有 3D 打印文件、组装说明和控制器软件均开源。

Partial Motion Imitation for Learning Cart Pushing with Legged Manipulators
arXiv 2603.26659 · 2026-03-27 Robotics 腿式操作
提出局部运动模仿方法，将运动任务中学到的步态风格迁移到推车操作场景。先通过大量域随机化和地形随机化训练鲁棒运动策略，再用局部对抗运动先验（partial adversarial motion prior）仅模仿下半身运动来学习 loco-manipulation 策略。在 IsaacLab 中成功沿多样轨迹推车，并有效迁移到 MuJoCo，相比基线实现更稳定、更精确的腿式操作行为。

Learning to Commit: Generating Organic Pull Requests via Online Repository Memory
arXiv 2603.26664 · 2026-03-27 Agent 代码生成
提出 Learning to Commit 框架，通过在线仓库记忆（Online Repository Memory）让 LLM 编码智能体生成更"有机"的 Pull Request。核心发现：现有编码智能体的 PR 常被拒绝不是因为功能错误，而是忽略了项目特定的约定、重复内部 API 或违反隐含架构约束。该框架让智能体对历史提交进行监督对比反思——尝试盲目解决历史问题，与真实差异对比，提炼为可复用的"技能"（编码风格、内部 API 用法、架构不变量）。新 PR 生成时基于累积技能调节输出，在未来已合并的 PR 上验证有效提升有机性评分。

Vision2Web: A Hierarchical Benchmark for Visual Website Development with Agent Verification
arXiv 2603.26648 · 2026-03-27 Agent 网站开发
面向视觉网站开发的分层基准 Vision2Web，覆盖从静态 UI 到代码生成、交互式多页前端复刻、到长视距全栈开发三个层级。基于真实网站构建，包含 193 个任务（16 个类别）、918 个原型图像和 1,255 个测试用例。提出基于工作流的代理验证范式，含 GUI 代理验证器和 VLM 判断器两个互补组件。评估显示 SOTA 模型在全栈开发方面仍存在巨大性能差距。

Weight Tying Biases Token Embeddings Towards the Output Space
arXiv 2603.26663 · 2026-03-27 LLM 嵌入分析
揭示权重绑定（Weight Tying）的隐含偏差机制：共享的嵌入矩阵主要为输出预测优化，而非输入表示。原因在于输出梯度在训练早期占主导地位，导致"反嵌入偏差"——早期层计算效率降低，对残差流贡献变弱。通过缩放输入梯度可减少此偏差，提供了因果证据。这些发现有助于解释为什么权重绑定在模型规模扩大时损害性能，对训练小型 LLM（嵌入占参数比例大）尤其有启示。

1 篇
Make Geometry Matter for Spatial Reasoning
arXiv 2603.26639 · 2026-03-27 Multimodal 空间推理
提出 GeoSR 框架，让视觉语言模型积极利用几何 Token 进行空间推理。两大核心组件：几何释放掩码在训练期间策略性屏蔽部分 2D 视觉 Token，削弱非几何捷径，迫使模型咨询几何 Token；几何引导融合通过门控路由机制自适应放大几何证据关键区域的几何 Token 贡献。在静态和动态空间推理基准上建立新 SOTA，有效克服现有方法中模型过度依赖 2D 视觉线索而忽视几何线索的问题。

Beyond Language: Grounding Referring Expressions with Hand Pointing in Egocentric Vision
arXiv 2603.26646 · 2026-03-27 Other 手势定位
首个面向自我中心视角的手势指向视觉定位大规模多模态数据集 EgoPoint-Ground，包含复杂场景中超过 15k 个交互样本，提供手-目标框对和密集语义标注。提出 SV-CoT 基线框架，通过视觉思维链范式将定位重构为结构化推理——协同手势和语言线索，比现有方法实现 11.7% 绝对提升。有效缓解语义歧义，提升理解多模态物理意图的能力。
