🔥 UniVidX: A Unified Multimodal Framework for Versatile Video Generation via Diffusion Priors
arXiv 2605.00658 · 2026-05 · HuggingFace Community
提出统一的多模态视频生成框架 UniVidX，基于扩散先验实现多种视频生成任务的统一处理。支持文生视频、图生视频、视频编辑等多种模态组合，通过共享扩散模型和灵活的条件注入机制，在单一框架内覆盖多样化的视频生成需求。

End-to-End Autoregressive Image Generation with 1D Semantic Tokenizer
arXiv 2605.00503 · 2026-05 · ByteDance Seed
自回归图像建模依赖视觉 Tokenizer，本文提出 1D 语义 Tokenizer，实现端到端自回归图像生成。相比传统 2D 离散 Token 方案，1D 语义表示更紧凑高效，在保持生成质量的同时显著提升推理速度和序列压缩率。

Let ViT Speak: Generative Language-Image Pre-training (GenLIP)
arXiv 2605.00809 · 2026-05 · ByteDance
提出生成式语言-图像预训练框架 GenLIP，让 ViT 具备「说话」能力。通过将视觉编码器与语言生成器统一训练，实现视觉特征到自然语言的直接生成。核心创新在于打破传统 CLIP 式对比学习的判别范式，转向生成式预训练，使视觉模型获得更丰富的语义理解能力。

💬 LLM · 大语言模型

Themis: Training Robust Multilingual Code Reward Models for Flexible Multi-Criteria Scoring
arXiv 2605.00754 · 2026-05 · Themis Team
Reward Models (RMs) 已成为 LLM 对齐训练中不可或缺的组件。Themis 聚焦多语言代码奖励模型的鲁棒训练，支持灵活的多维度评分标准。解决了现有代码 RM 在跨语言泛化和多标准评估上的不足，为代码生成的 RLHF 训练提供更可靠的奖励信号。
