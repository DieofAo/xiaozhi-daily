HOTTAMEn: Tactile-Aware Manipulation Engine for Closed-Loop Data Collection in Contact-Rich Tasks
Robotics触觉操作
提出触觉感知的操作引擎 TAMEn，用于接触密集任务的闭环数据采集。核心包括：跨形态可穿戴接口适配不同夹爪，双模态采集管线结合动捕精密模式与 VR 便携模式，以及金字塔数据体制统一大规模触觉预训练、双臂演示和人在回路恢复数据。基于视觉-触觉学习框架，双臂操作成功率从 34% 提升至 75%。硬件和数据集已开源，为触觉驱动的操作数据规模化提供了实用基础设施。

HOTRoSHI: A Versatile Robot-oriented Suit for Human Data In-the-Wild
Robotics人体动捕
面向机器人学习的混合可穿戴动捕系统。融合低成本稀疏 IMU 与 Project Aria 智能眼镜第一人称视角，在全局度量坐标系下估计穿戴者完整 3D 姿态和体型。IMU 提供抗遮挡与高速运动鲁棒性，第一人称 SLAM 锚定长时运动并稳定上半身估计。采集的运动数据已验证可用于真实世界人形机器人策略学习。规模化人体运动数据采集的实用方案。

NEWRobots that Learn to Evaluate Models of Collective Behavior
Robotics仿生交互
基于强化学习的仿生机器鱼框架，通过与真实鱼群的闭环交互来评估计算行为模型。在仿真中训练四种不同保真度的鱼类模型（简单基准、规则模型、CNN 模型），迁移至真实机器鱼设置。通过量化 Sim-to-Real gap，发现神经网络模型在目标达成和行为指标上保真度最高。展示了具身交互如何系统性地揭示行为模型的缺陷——一个评估动物集体行为模型的通用框架。

NEWPersonalized RewardBench: Evaluating Reward Models with Human Aligned Personalization
LLM个性化对齐
首个评估奖励模型个性化偏好建模能力的基准。通过构建基于特定用户标准的选择/拒绝响应对，确保偏好区分是个人化的而非通用的。测试发现当前 SOTA 奖励模型在个性化方面表现不佳，准确率峰值仅 75.94%。研究还证明该基准与下游任务（Best-of-N 采样和 PPO）的性能呈更高相关性。多元化对齐是 LLM 下一阶段的关键前沿。

NEWSyntax Is Easy, Semantics Is Hard: Evaluating LLMs for LTL Translation
LLM形式化验证
系统评估 LLM 将自然语言翻译为命题线性时序逻辑（LTL）公式的能力。发现 LLM 在语法层面表现良好，但语义理解是瓶颈——生成的 LTL 公式语法正确率高但语义等价率显著更低。揭示了 LLM 在形式化规范生成中"形似而神不至"的局限性，对机器人任务规范自动化有直接警示意义。

NEWEvaluating In-Context Translation with Synchronous Context-Free Grammar Transduction
LLM低资源翻译
利用同步上下文无关文法（SCFG）转导来评估 LLM 的上下文内翻译能力，聚焦低资源语言。研究发现 LLM 在利用上下文示例时存在系统性偏差——倾向于过度依赖表面形式匹配而忽略结构级对应关系。为理解和改进 LLM 少样本翻译机制提供了形式化分析框架。
