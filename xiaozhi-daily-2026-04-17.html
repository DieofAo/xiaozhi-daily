π0.7: A Steerable Generalist Robotic Foundation Model with Emergent Capabilities Boss Pick
VLA Foundation Model
Physical Intelligence 发布的通用机器人基础模型 π0.7，被称为"机器人领域的 GPT-3 时刻"。核心突破在于组合泛化（Compositional Generalization）——模型首次展示了将不同场景学到的技能重新组合解决未见任务的能力。架构包含 VLA 主模型 + 动作专家 + 高层策略 + 轻量级世界模型（生成视觉子目标），通过多模态提示框架（语言指令 + 元数据 + 控制模态 + 视觉子目标）统一异构训练数据。关键结果：(1) 单一通用模型在折衣、做咖啡、折箱等任务上匹配/超越各任务独立 RL 专家模型 π*0.6；(2) UR5e 双臂零样本跨形态叠衣服，成功率与专家遥操作者首次使用该硬件相当；(3) 空气炸锅烤红薯仅凭 2 条间接相关数据 + 语言教练即可完成；(4) 能力增长呈超线性扩展。基于 Gemma3 4B，VLA 推理单张 H100 可达 38ms。80+ 作者包含 Chelsea Finn、Sergey Levine 等。公司估值 $56B，传闻新一轮融资估值 $11B。

World-Value-Action (WAV) Model: Implicit Planning for VLA via World Model and Value Function Boss Pick
VLA World Model
提出 WAV 模型，将世界模型、轨迹价值函数和动作生成统一在一个框架中。世界模型预测未来状态，价值函数评估长期效用，动作生成被形式化为在潜在空间中的推理过程。与显式规划不同，WAV 实现了隐式规划——在推理时在潜在空间中搜索高价值动作序列。在 CALVIN 和 SimplerEnv 基准上显著超越现有 VLA 基线。

VLA 模仿学习

R3D: Revisiting 3D Policy Learning — Scaling Transformer-based 3D Diffusion Policies
VLA 3D 策略

RL 双足运动
