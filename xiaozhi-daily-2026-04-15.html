🤖 VLA · Vision-Language-Action（4 篇）

VGA: Vision-Geometry-Action Mapping for Robotic Manipulation
VLA 操作范式
提出全新的"视觉→几何→动作"操作范式，通过中间几何表征桥接视觉感知与机器人动作生成。核心思路：先从视觉观察构建 3D 几何理解，再基于几何约束生成精确的操作动作序列。在多种操作任务上显著优于端到端 VLA 方法，特别是在需要精确空间推理的任务中表现突出。
arXiv PDF
HazardArena: A Benchmark for Evaluating Semantic Safety in VLA Models
VLA 安全评估
首个针对 VLA 模型语义安全性的系统化评估基准。在多种真实场景中测试 VLA 是否会执行不安全的指令（如将刀递向人脸、忽略障碍物等），揭示当前主流 VLA 模型在安全方面存在显著盲区。提出一套安全评价指标体系和防御增强策略。
arXiv PDF
TAX-DPD: Task-Aware Cross-Domain Disentangled Point Diffusion for Object Placement
VLA 精确放置
提出任务感知跨域解耦点扩散方法，用于精确的物体放置任务。将放置问题分解为目标位姿估计和放置轨迹规划两个解耦子问题，通过点云扩散模型实现跨域泛化。在新物体、新场景的零样本放置任务中展现强泛化能力。
arXiv PDF
DeCoNav: Decoupled Learning for Cooperative Visual Language Navigation
VLA 协作导航
提出解耦学习框架用于多智能体协作视觉语言导航，将全局规划与局部执行解耦。解决了多智能体 VLN 中的通信开销和协调难题，在 REVERIE 和 CVDN 基准上取得 SOTA。
arXiv PDF

🦾 Robotics（5 篇）

Touch Dreaming: Enabling Humanoid Robot Manipulation with Tactile Sensing
Robotics 触觉感知
为人形机器人配备高密度触觉传感，实现从"看到就抓"到"摸到才精"的操作升级。将触觉信号深度集成到操作策略中，人形机器人在精细操作（如组装小零件、拿取易碎物品）中成功率提升 35%+。开源触觉-操作联合数据集。
arXiv PDF
XRZero-G0: A Dexterous Manipulation Demonstration Data Collection System
Robotics 数据采集
基于 XR（扩展现实）技术构建零重力环境下的灵巧操作示教数据采集系统。结合手部追踪和力反馈手套，操作员可直觉式示教复杂灵巧动作。高效采集大规模高质量示教数据，为灵巧手策略学习提供可扩展数据管线。
arXiv PDF
GraG: Fast Hand-Object 3D Reconstruction with Grasp Geometry
Robotics 3D 重建
利用抓取几何先验实现快速手-物体联合 3D 重建。关键洞察：手部抓取约束提供了强大的形状和位姿先验，可大幅减少重建所需的观察数量。从单张 RGB 图像在 50ms 内完成手-物体联合重建，精度接近多视角方法。适用于机器人抓取规划和操作仿真。
arXiv PDF
FastGrasp: Efficient Whole-body Fast Dexterous Grasping for Humanoid Robots
Robotics 灵巧抓取
实现人形机器人全身协调快速灵巧抓取。将抓取规划和全身运动控制统一优化，使人形机器人能在 <1s 内完成从感知到执行的全链路。关键技术：层次化策略（全身姿态→手臂轨迹→灵巧手闭合）+ 实时碰撞规避。在 Unitree H1 上实现真机验证。
arXiv PDF
AutoMoMa: Scalable Trajectory Generation for Mobile Manipulators
Robotics 移动操作
面向移动操作机器人的可扩展轨迹生成框架。自动化处理底盘运动+机械臂操作的联合规划，支持大规模并行生成高质量轨迹数据。为移动操作策略的数据驱动训练提供可扩展基础设施。在多种家庭和工业场景中验证。
arXiv PDF

🌍 Embodied AI（3 篇）

WHOLE-MoMa: Offline Reinforcement Learning for Whole-Body Mobile Manipulation
Embodied AI 离线 RL
首个将离线 RL 系统化应用于全身移动操作的工作。核心挑战：mobile manipulation 涉及底盘移动+臂操作+全身协调的高维联合动作空间，传统在线 RL 探索效率极低。WHOLE-MoMa 从离线数据集高效学习全身协调策略，包含分层状态表征和约束感知的策略优化。在模拟和真机上验证，复杂长程操作任务成功率显著提升。
arXiv PDF
Tree Learning: Multi-Skill Continual Learning for Humanoid Robots
Embodied AI 持续学习
为人形机器人设计的多技能持续学习框架。核心思想：将技能组织为树状结构，共享底层运动原语（locomotion primitives）并逐步分化高层操作技能。有效解决灾难性遗忘问题，每次学新技能时旧技能保持率 >95%。在 12+ 种人形运动和操作技能上验证。
arXiv PDF
Learning Contact-Rich Manipulation from Large-Scale Human Video
Embodied AI 视频学习
从大规模人类视频中学习接触丰富的操作技能。利用互联网级人类操作视频作为隐式先验，通过视觉对比学习提取操作几何和力交互模式。零样本迁移到机器人平台，在门把手操作、抽屉开关等接触密集任务上表现出色。
arXiv PDF

🎯 RL · Reinforcement Learning（2 篇）

Relax: An Open-Source RL Training Engine by REDnote
RL 训练框架
小红书开源 RL 训练引擎 Relax（4 月 15 日发布）。专为大模型 RLHF/PPO/DPO 训练优化，支持分布式异步训练和自适应 KL 散度控制。核心优势：训练效率比 TRL 提升 3-5 倍，支持 70B+ 参数模型在 8 卡 A100 上训练。已开源完整代码和训练 recipe。
GitHub
GigaWorld-Policy: 10x More Efficient Training for Embodied Agents
RL 训练效率
极佳视界（4 月 14 日）发布 GigaWorld-Policy，核心卖点是具身智能训练效率 10 倍提升。通过世界模型辅助的策略学习 + 高效并行仿真环境，将原本需要数天的训练压缩到数小时。在 manipulation 和 locomotion 任务上验证效果。
Official

🌐 World Model（2 篇）

Genie Envisioner 2.0: Interactive World Model by AgiBot
World Model 交互式
智元机器人发布 Genie Envisioner 2.0 交互式世界模型。支持用户输入动作指令后生成对应的视觉预测结果，实现"想象→验证→执行"的规划闭环。在机器人操作和导航任务中作为策略的辅助评估工具，减少真实试错成本。结合 AgiBot 的硬件平台实现端到端验证。
Official
Efficient World Models with Context-Aware Tokenization
World Model 高效建模
提出上下文感知 Token 化方法提升世界模型效率。关键洞察：环境中不同区域的信息密度差异巨大，对高动态区域使用更多 token、对静态背景使用更少 token。在 Atari 和机器人仿真环境上，相同参数量下预测精度提升 20%+，推理速度提升 3 倍。
