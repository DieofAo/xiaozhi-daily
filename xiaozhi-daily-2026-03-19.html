NEW 6-DOF Multimodal Transformer

GMT: Goal-Conditioned Multimodal Transformer for 6-DOF Object Trajectory Synthesis in 3D Scenes

TU Munich · arXiv 2603.17993 · 2026-03-18
合成可控的 6-DOF 物体操控轨迹是机器人与复杂场景交互的核心能力。现有方法通常依赖 2D 或部分 3D 表示，难以捕捉完整场景几何。GMT 提出了一种多模态 Transformer 框架，从 3D 场景中直接生成目标条件下的 6 自由度物体操控轨迹，具备精确的空间推理能力和物理可行性。在多样化 3D 环境中展示了出色的轨迹合成精度。

Shape Sensing OFDR

Single-Fiber OFDR-Based Shape Sensing of Concentric Tube Steerable Drilling Robots

arXiv 2603.17990 · 2026-03-18
提出一种基于光频域反射仪（OFDR）的同心管可转向钻进机器人形状感知方法。与传统 FBG 方法不同，OFDR 能沿光纤全长进行连续应变测量，空间分辨率更高。设计了形状感知组件（SSA），将单根 OFDR 光纤与平 NiTi 线集成，实现了对细长柔性机器人体的精确形状重建，为医疗/工业场景下的柔性机器人提供了实用传感方案。

亮点 Token Pruning Video VLM

Unified Spatio-Temporal Token Scoring for Efficient Video VLMs

arXiv 2603.18004 · 2026-03-18
Token 剪枝是提升视觉语言模型（VLM）计算效率的关键，在视频任务中时间冗余尤为突出。现有方法要么仅在 ViT 内部针对单模态感知任务剪枝（不适配下游 VL 任务），要么仅在 LLM 内剪枝却保留 ViT 输出完整。本文提出统一的时空 Token 评分机制，同时在 ViT 和 LLM 层面实现高效 Token 管理，在保持模型精度的同时显著降低视频 VLM 的计算开销。

Skeleton Differentiable Rendering

Universal Skeleton Understanding via Differentiable Rendering and MLLMs

arXiv 2603.18003 · 2026-03-18
多模态大语言模型（MLLM）虽然视觉-语言推理能力强大，但无法直接处理骨架这类结构化非视觉数据。现有方法要么将骨架动态压缩为有损特征向量做文本对齐，要么将运动量化为离散 Token（跨异构骨架格式泛化性差）。SkeletonLLM 通过可微渲染将任意骨架格式转换为视觉表示，实现了通用的骨架理解能力。

亮点 Self-Evolution Code-as-Experience

AgentFactory: A Self-Evolving Framework Through Executable Subagent Accumulation and Reuse

arXiv 2603.18000 · 2026-03-18
构建基于 LLM 的智能体日益重要。现有 Agent 自演进方法主要将成功经验记录为文本提示或反思，这在复杂场景中无法可靠保证高效重执行。AgentFactory 提出全新的自演进范式：将成功的任务解决方案保存为可执行的子 Agent 代码（而非文本经验），并基于执行反馈持续优化这些子 Agent。这些积累的子 Agent 可以在新任务中被发现和复用，形成不断壮大的能力库。
