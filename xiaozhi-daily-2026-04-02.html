Universal YOCO for Efficient Depth Scaling
LLM 高效架构
提出 Universal YOCO（You Only Cache Once）架构的升级版本，实现更高效的深度扩展。随着 test-time scaling 的兴起，推理和 agentic 能力对模型深度提出了更高要求。该方法通过"只缓存一次"的思想优化 KV-Cache，使得大模型在推理时的内存开销显著降低，同时保持性能。对长上下文和复杂推理场景尤为重要。

RELISH: LLM REgression with a Latent Iterative State Head
LLM 回归预测
提出 RELISH（REgression with a Latent Iterative State Head），一种新型的轻量级方法让大语言模型进行数值回归预测。传统 LLM 以文本 token 形式输出数字，精度和泛化性受限。RELISH 在模型头部引入迭代式隐状态机制，使 LLM 能以更自然的方式进行连续值预测，在多个回归任务上超越传统方法。
