🌍 World Model · 世界模型

🔥 LingBot-World: 开源实时交互世界模型
Robbyant (蚂蚁集团) · 2026-05 · 开源发布
蚂蚁集团具身智能公司 Robbyant 开源 LingBot-World，28B 参数的动作条件化视频世界模型。核心亮点：实时 16 FPS / 720p 输出、端到端延迟 <1s、支持 10+ 分钟时序一致性。不同于被动视频生成，LingBot-World 创建可交互世界——支持键盘、相机运动和语言动作条件控制。适用于具身智能训练、自动驾驶仿真和游戏开发。Apache 许可证开源。

World2VLM: Distilling World Model Imagination into VLMs for Dynamic Spatial Reasoning
arXiv 2604.26934 · 2026-04-29 · 中科院自动化所
提出将生成式世界模型的空间想象能力蒸馏到 VLM 中的训练框架。VLM 在静态视觉理解上表现优异，但在需要想象场景如何在自我运动下演变的动态空间推理中仍有困难。World2VLM 通过轨迹生成 + 监督构建的多阶段 pipeline，让 VLM 获得「想象未来场景」的能力，显著提升动态空间推理性能。

🧠 RL · 强化学习

🔥 ARTIST: Agentic Reasoning and Tool Integration for LLMs via Reinforcement Learning
arXiv 2505.00024 / 2505.01441 · 2026-05 · NVIDIA
NVIDIA 提出统一框架将 Agent 推理、RL 和工具集成紧密结合。模型自主决定在多轮推理链中何时/如何/调用哪些工具，通过基于结果的 RL 学习强大的工具使用策略，无需步骤级监督。在复杂数学推理和多轮函数调用任务上效果提升高达 22%，显著减少幻觉并提高工具使用准确率。
RL LLM Tool Use

DORA: A Scalable Asynchronous Reinforcement Learning System for Language Model Training
arXiv 2604.26256 · 2026-04-29 · 美团
针对 LLM RL 训练中 rollout 阶段（占 50-80% 步骤时间）被长尾生成瓶颈的问题，提出完全异步的 RL 系统 DORA。将生成与训练彻底解耦到不同 GPU 集群并行执行，配合多版本流式训练和权重过时管理策略，在相同 GPU 资源下实现高效扩展，解决了长序列生成导致的同步阻塞问题。

Addressing Performance Saturation for LLM RL via Precise Entropy Curve Control
arXiv 2604.26326 · 2026-04-29 · Purdue University
发现并系统解决 LLM RL 训练中的性能饱和问题。通过精确控制策略熵曲线演化，避免 RL 训练过早收敛到次优解。提出自适应熵控制机制，让模型在保持探索多样性的同时稳步提升推理能力。实验表明在数学推理任务上突破了传统 RL 方法的性能天花板。

💬 LLM · 大语言模型

ProRL: 延长强化学习突破大语言模型推理极限
arXiv 2026 · NVIDIA · HuggingFace 开源权重
通过实验证明延长 RL（ProRL）训练可以发现基础模型无法触及的新型推理策略。提出结合 KL 散度控制、参考策略重置和多样化任务套件的新训练方法。关键发现：RL 训练的模型在各种 pass@k 评估中始终优于基础模型，包括基础模型无论尝试多少次都完全失败的场景。证明 RL 能探索和填充解空间的新区域。

M3-Agent: Multimodal Agent with Long-Term Memory
ICLR 2026 · ByteDance Seed · 开源
