🤖 VLA · Vision-Language-Action（7 篇）

StarVLA-α: Reducing Complexity in Vision-Language-Action Systems
VLA 简洁设计
提出简洁而强大的 VLA 基准 StarVLA-α，刻意最小化架构和管道复杂性以减少实验混杂因素。系统重新评估动作建模策略、机器人预训练和接口工程等关键设计轴。在 LIBERO、SimplerEnv、RoboTwin 和 RoboCasa 统一多基准上保持高度竞争力，表明强大 VLM 主干加最小设计已足够。单一通用模型在 RoboChallenge 上比 π₀.₅ 高 20%。
arXiv PDF GitHub
DA-PTQ: Drift-Aware Post-Training Quantization for Efficient VLA Models
VLA 模型压缩
首次识别 VLA 模型量化中的时间误差累积问题——量化扰动在视觉-语言-动作接口处逐步放大，导致执行轨迹的运动学漂移。提出 DA-PTQ 方法，包含跨空间表示补偿和运动驱动的混合精度分配两个组件，将量化表述为序列决策中的漂移感知优化。低比特设置下达到全精度模型相当性能。
arXiv PDF
AnySlot: Goal-Conditioned VLA Policies for Zero-Shot Slot-Level Placement
VLA 精密放置
通过引入显式空间视觉目标作为中间表征，解决复合语言指令下的精确物体放置挑战。分层设计将高级槽位选择与低级执行分离，确保语义准确性和空间鲁棒性。引入 SlotBench 基准（9 个任务类别），零样本性能显著超过平面 VLA 基线。
arXiv PDF
LARY: Latent Action Representation Benchmark for Vision-to-Action Alignment
VLA 表示学习
建立评估潜在动作表示的统一基准框架，包含 100 万+ 视频（1000 小时）、151 个动作类别、62 万图像对和 59.5 万条动作轨迹。关键发现：未经动作监督训练的通用视觉基础模型一致优于专门的具身潜在动作模型；基于潜在视觉空间的表示比像素空间更好地对齐物理动作空间。
arXiv PDF
CLAW: Composable Language-Annotated Whole-body Motion Generation
VLA 人形运动
为 Unitree G1 人形机器人可扩展生成语言标注全身运动数据。开发交互式网络平台，包含可组合运动规划（方向/速度/高度参数化）、实时键盘和时间轴编辑、MuJoCo 仿真控制和自动标注引擎。支持 50Hz 物理可行轨迹采样和多样化语言描述风格。已开源。
arXiv PDF
OmniUMI: Physically Grounded Robot Learning via Human-Aligned Multimodal Interaction
VLA 触觉+力
统一框架同步采集 RGB、深度、轨迹、触觉、内部抓握力和外部交互力。通过共享 embodiment 设计保持采集-部署一致性，提供双向夹爪力反馈。扩展扩散策略加入视觉/触觉/力观察模块，通过阻抗控制统一调节运动和接触行为。在力敏感拾取、交互式擦除和触觉引导释放任务上验证。
arXiv PDF
WARPED: Wrist-Aligned Rendering for Robot Policy Learning from Human Demos
VLA 人类演示
从自中心 RGB 视频合成逼真腕部视图观测来训练机器人策略。使用视觉基础模型初始化交互场景，追踪手部和物体，重定向到机器人末端执行器，通过高斯溅射合成渲染。在 5 项桌面操纵任务中达到与遥操作数据训练相当的成功率，数据收集时间减少 5-8 倍。
arXiv PDF

🌍 World Model（4 篇）

Grounded World Model for Semantically Generalizable Planning
World Model 语义泛化
在视觉-语言对齐隐空间中学习世界模型，将 MPC 转化为 VLA。通过行动未来结果与任务指令的相似度评分，实现对未见视觉信号和新表达方式的语义泛化。WISER 基准测试集成功率 87%，而传统 VLA 仅 22%（尽管在训练集上达 90%），展示了显著的泛化优势。
arXiv PDF
WM-DAgger: Efficient Data Aggregation for Imitation Learning with World Models
World Model 模仿学习
利用世界模型合成 OOD 恢复数据，无需人工干预解决模仿学习复合误差问题。包含纠正动作合成模块（防止误导性监督）和一致性引导过滤模块（筛除物理不合理轨迹）。少样本设置下（仅 5 个演示），软包推动任务成功率达 93.3%。
arXiv PDF
AIM: Intent-Aware Unified World Action Modeling with Spatial Value Maps
World Model 价值预测
预测对齐的空间价值图作为显式中间表征，弥合预训练视频模型与机器人控制的差距。采用混合 Transformer + 意图因果注意力 + 自蒸馏 RL。30K 操纵轨迹数据集上 RoboTwin 2.0 基准平均成功率 94.0%，在长地平线和接触敏感任务上尤为突出。
arXiv PDF
3D-Anchored Lookahead Planning via World-Model-Based MCTS
World Model 场景记忆
将 MCTS 与 3D 一致世界模型结合，维持持久 c2w 锚点实现遮挡下存活和重新规划。5 步顺序到达任务（需空间记忆）成功率 0.650，贪婪基线仅 0.006（Δ=+0.645）。消融研究：树搜索空间记忆贡献 82% 收益，更深前瞻贡献 17%。
arXiv PDF

🦾 Robotics（4 篇）

AffordSim: Scalable Data Generator for Affordance-Aware Robotic Manipulation
Robotics 可供性
首个将开放词汇 3D 可供性预测集成到操纵数据生成管道的仿真框架。基于 NVIDIA Isaac Sim，支持 Franka FR3/Panda/UR5e/Kinova，含 VLM 驱动任务生成和 3D 高斯重建域随机化。50 任务基准（7 类）：抓取 53-93% 成功率，但倒入窄容器（1-43%）和杯挂（0-47%）仍是挑战。零样本 Sim2Real 在 Franka FR3 上验证。
arXiv PDF
ComSim: Scalable Real-World Robot Data Generation via Compositional Simulation
Robotics Sim2Real
提出组合式模拟，结合经典和神经模拟生成准确的动作-视频对。采用闭环 real-sim-real 数据增强管道，训练神经模拟器将经典模拟视频转换为真实世界表示，显著降低 sim2real 域间隙。从小规模真实数据出发生成大规模多样化训练数据，提高策略成功率。
arXiv PDF
ViserDex: Visual Sim-to-Real for Robust Dexterous In-hand Reorientation
Robotics 灵巧手
利用 3D 高斯溅射（3DGS）弥合单目 RGB 灵巧手操作的视觉 Sim2Real 差距。在高斯表示空间中进行物理一致的预渲染增强生成域随机化数据，结合课程 RL + 师生蒸馏。消费级硬件可独立训练，在 5 种物体上成功验证重定向，即使在挑战性光照下也能工作。
arXiv PDF
AffordGen: Diverse Demonstrations for Generalizable Manipulation with Afford Correspondence
Robotics 零样本泛化
利用 3D 生成模型和视觉基础模型的语义对应关系，从大规模 3D 网格中生成新操纵轨迹。训练闭环视觉运动策略，将可供性语义泛化与端到端学习的鲁棒性结合。在模拟和真实世界中达到高成功率，对完全未见物体实现零样本泛化。
arXiv PDF

🏠 Embodied AI（2 篇）

EmbodiedGovBench: Governance, Recovery, and Upgrade Safety in Embodied Agent Systems
Embodied AI 安全治理
提出面向治理的具身代理评估基准，超越传统任务成功率指标。涵盖 7 个治理维度：未授权能力调用、运行时漂移鲁棒性、恢复成功、策略可移植性、版本升级安全性、人类覆盖响应性和审计完整性。跨越单机器人和集群设置，定义了场景模板、扰动算子和治理指标。
arXiv PDF
Ro-SLM: Onboard Small Language Models for Robot Task Planning (ACL 2026)
Embodied AI 小模型
通过蒸馏 LLM 知识实现 SLM 驱动的板载机器人操作，解决云端依赖和计算受限问题。利用 LLM 生成多样化任务指令和真值代码，LLM 作为奖励函数指导 SLM 微调。在无人机操作任务上从无法支持到实现接近 LLM 性能。ACL 2026 录用。
arXiv PDF

🎯 RL · 强化学习（1 篇）

ScoRe-Flow: Score-Based RL for Flow Matching Robot Control
RL 流匹配
提出基于得分函数的 RL 微调方法，实现对 Flow Matching 策略随机转移均值和方差的解耦控制。通过得分函数调节漂移将探索引导向高概率区域，得分可从速度场闭式获得无需辅助网络。D4RL 运动任务比 SOTA 收敛快 2.4 倍，Robomimic 和 Franka Kitchen 成功率提高 5.4%。
