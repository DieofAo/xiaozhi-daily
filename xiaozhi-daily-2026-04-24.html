VLA NEW 04-24
LoHo-Manip: Long-Horizon Manipulation via Trace-Conditioned VLA Planning
Isabella Liu, An-Chieh Cheng, Rui Yan, Geng Chen, Ri-Zhao Qiu, Xueyan Zou, Sha Yi, Hongxu Yin, Xiaolong Wang, Sifei Liu
针对 VLA 在长程操作任务中的脆弱性，提出 LoHo-Manip 模块化框架。核心创新：用一个 Task-Management VLM 以滑动窗口方式管理子任务序列，生成 2D 关键点视觉轨迹（Visual Traces）作为局部控制目标，将长程决策拆解为"反复的局部控制"。隐式闭环机制：每步预测剩余计划，实现自动接续和重规划，无需手工恢复逻辑。在仿真和真实 Franka 机器人上展示了长程成功率、鲁棒性和 OOD 泛化的显著提升。

VLA NEW 04-24
ResVLA: From Noise to Intent — Anchoring Generative VLA Policies with Residual Bridges
Yiming Zhong, Yaoyu He, Zemin Yang, Pengfei Tian, Yifan Huang, Qingqiu Huang, Xinge Zhu, Yuexin Ma
提出 ResVLA，将 VLA 策略从"从噪声生成"范式转变为"从意图精修"范式。利用频谱分析将控制解耦为确定性低频锚点（全局意图）和随机高频残差（局部动力学），通过残差扩散桥（Residual Diffusion Bridge）精修局部动态。相比标准生成基线收敛更快、对语言和机器人具身扰动更鲁棒，并在真实机器人实验中验证。

VLA NEW 04-24
CorridorVLA: Explicit Spatial Constraints for Generative Action Heads via Sparse Anchors
Dachong Li, ZhuangZhuang Chen, Jin Zhang, Jianqiang Li
针对 VLA 模型中空间引导通常隐式嵌入潜在特征的问题，提出 CorridorVLA：预测稀疏空间锚点作为增量物理变化(Δ-positions)，为 flow-matching 动作头定义显式容差走廊（Corridor）——偏离走廊的轨迹接收校正梯度，轻微偏差被容忍。在 LIBERO-Plus 上取得 3.4%–12.4% 成功率提升，GR00T-Corr 变体达 83.21%。

Yupeng Zheng, Xiang Li, Songen Gu, Yuhang Zheng et al.

VLA NEW 04-24
How VLAs (Really) Work In Open-World Environments
Amir Rasouli, Yangzheng Wu, Zhiyuan Li, Rui Heng Yang, Xuan Zhao, Charles Eret, Sajjad Pakdamansavoji
对 SOTA VLA 模型在 BEHAVIOR1K 基准上的深入批判性分析。发现当前评估指标（成功率/部分分数）仅关注最终物体状态，忽视过程和安全性，可能夸大性能。提出新的多维评估协议，覆盖可复现性、安全性、任务感知和失败因素分析，为 VLA 的真实部署提供更可靠的评估参考。

Hi-WM: Human-in-the-World-Model for Scalable Robot Post-Training
Yaxuan Li, Zhongyi Zhou, Yefei Chen, Yanjiang Guo, Jiaming Liu, Shanghang Zhang, Jianyu Chen, Yichen Zhu
提出 Human-in-the-World-Model (Hi-WM)，将世界模型作为可复用的人类修正基底，实现无需物理执行的策略后训练。支持中间状态缓存、回滚和分支，一个失败状态可产生多条修正轨迹，生成密集的局部监督信号。关键结果：真实世界成功率比基策略提升 37.9 个百分点，比世界模型闭环基线高 19.0 个百分点，世界模型评估与真实表现高度相关 (r = 0.953)。

Agent MCP
Tool Attention Is All You Need: Dynamic Tool Gating and Lazy Schema Loading for Eliminating the MCP/Tools Tax
Anuj Sadani, Deepak Kumar
直击 LLM Agent 工作流中的痛点——MCP 协议的"工具税"（每轮 10k–60k tokens 开销）。提出 Tool Attention 中间件：用 Intent Schema Overlap (ISO) 评分+状态感知门控实现动态工具选择，双阶段懒加载 schema。核心结果：每轮工具 token 减少 95%（47.3k→2.4k），有效上下文利用率从 24% 提升至 91%。已开源。
