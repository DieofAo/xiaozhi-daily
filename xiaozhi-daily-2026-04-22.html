VLA Foundry: A Unified Framework for Training Vision-Language-Action Models
VLA 统一框架
提出开源框架 VLA Foundry，在单一代码库中统一 LLM、VLM 和 VLA 训练全流程。支持从零开始训练与 Hugging Face 预训练骨干（如 Qwen3-VL）微调两种模式。在 LBM Eval 仿真评估中，Qwen3-VL 骨干的多任务桌面操作策略表现优异。32 页技术报告，完整开源。

UniT: Toward a Unified Physical Language for Human-to-Humanoid Policy Learning and World Modeling
VLA 人形机器人 Boss 关注
建立人类到人形机器人的统一物理语言框架 UniT。采用三分支交叉重建机制（动作+视觉+融合）形成具身不可知的物理意图离散潜在空间。实现 VLA-UniT 策略学习和 WM-UniT 世界建模两大应用，充分利用大规模人类运动数据进行人形机器人迁移学习。来自小鹏机器人。

LiveVLN: Breaking the Stop-and-Go Loop in Vision-Language Navigation
VLA 导航连续性
提出无训练框架 LiveVLN，为预训练 VLM 导航器添加多步动作连续性。通过执行与新观测处理时间重叠，消除传统感知-推理-执行循环的阻塞等待。在 R2R 和 RxR 基准上保持性能，真实部署中将等待时间减少 77.7%。

🌍 World Model · 世界模型 2 篇
Mask World Model: Predicting What Matters for Robust Robot Policy Learning
World Model 鲁棒策略
采用视频扩散架构预测语义掩码而非像素，通过几何信息瓶颈捕捉本质物理动力学和接触关系，过滤动态背景与照明变化等视觉噪声。在 LIBERO 和 RLBench 基准上显著超越最先进的 RGB 世界模型，真实机器人实验中展现优越的纹理鲁棒性和泛化能力。
arXiv PDF
RoboWM-Bench: A Benchmark for Evaluating World Models in Robotic Manipulation
World Model 评估基准
提出以操纵为中心的视频世界模型评估基准 RoboWM-Bench。将人手和机器人操纵视频中的生成行为转换为具身动作序列并通过机器人执行验证。发现常见失败模式包括空间推理错误、接触预测不稳定和非物理变形，物理不一致问题即使微调后仍存在。

GenerativeMPC: VLM-RAG-guided Whole-Body MPC with Virtual Impedance for Bimanual Mobile Manipulation
Whole-Body MPC 双臂移动操纵 Boss 关注
提出分层网络物理框架 GenerativeMPC，显式将语义场景理解与物理控制参数关联。VLM-RAG 将视觉和语言转化为接地控制约束，为全身 MPC 输出动态速度限制和安全边距。虚拟阻抗-导纳控制实现人机交互中的上下文感知柔顺。在 MuJoCo、IsaacSim 和物理平台验证，人类附近实现 60% 速度降低。

Scalable Trajectory Generation for Whole-Body Mobile Manipulation (AutoMoMa)
Mobile Manipulation 轨迹生成 Boss 关注
GPU 加速框架 AutoMoMa，统一 AKR 建模（将基座、臂和物体运动学整合为单一链）与并行化轨迹优化。达到每 GPU 小时 5000 个轨迹（比 CPU 快 80 倍），生成 500K+ 物理有效轨迹数据集，覆盖 330 个场景和多个机器人具身。同时解决规模、多样性和运动学保真度三大问题。
arXiv PDF
HEX: Humanoid-Aligned Experts for Cross-Embodiment Whole-Body Manipulation
Whole-Body Control VLA Boss 关注
针对全尺寸双足人形机器人协调操作的状态中心框架 HEX。引入人形对齐的通用状态表示用于跨异构体身可扩展学习，MoE 统一本体感觉预测器建模全身协调和时间运动动力学。轻量历史令牌 + 残差门控融合 + 流匹配动作头。在真实人形操作任务上 SOTA。17 位作者联合。

FastGrasp: Learning-based Whole-body Control for Fast Dexterous Grasping with Mobile Manipulators
灵巧抓取 RL Boss 关注
移动机器人快速灵巧抓取框架。两阶段 RL 策略：CVAE 根据物体点云生成多样化抓取候选，执行移动底座、臂和手部的协调运动。集成触觉反馈实时调整以处理冲击效应。在模拟和真实场景中实现有效 sim-to-real 转移，覆盖多样化物体几何形状。

Learning Hybrid-Control Policies for High-Precision In-Contact Manipulation Under Uncertainty
混合力/位控制 RL
学习混合位置-力控制策略，每个维度动态选择力或位置控制模式。提出 MATCH 方法调整策略动作概率以镜像混合控制模式选择。在脆弱 peg-in-hole 任务中：成功率提高 10%，钉子破损减少 5 倍。1600+ 仿真对比实验中高噪声环境成功率 68% vs 33%。
